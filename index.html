Bienvenido

Este sitio web forma parte de un proyecto colaborativo orientado al aprendizaje del desarrollo web y el uso de herramientas de control de versiones.

Introducción

Aprenderás los fundamentos de HTML, CSS y JavaScript para crear páginas web modernas e interactivas.

Propósito

Construir una base sencilla que evolucione con el trabajo del equipo, incorporando mejoras progresivas en diseño y funcionalidad.

Temas del Taller

  • HTML: estructura de páginas web
  • CSS: diseño y estilos
  • JavaScript: interactividad
  • GitHub: trabajo colaborativo
